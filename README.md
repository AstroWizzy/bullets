# bullets [![NPM version](https://badge.fury.io/js/bullets.svg)](http://badge.fury.io/js/bullets)

> Generate bullets for list items, or bulleted lists.

## Install with [npm](npmjs.org)

```bash
npm i bullets --save
```

## Usage

```js
var bullets = require('bullets');
```
